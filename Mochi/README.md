# Mochi

Fetch Mochi images

## Dependencies

The project uses [requests](https://pypi.org/project/requests).

## Installation

Set up virtual environment (VE):

```bash
python3 -m venv ./venv
```

Activate the VE:

```bash
source ./venv/bin/activate
```

Install dependencies

```bash
pip3 install -r ./requirements.txt
```

## Run

In an active VE, run the program

```bash
python main.py
```
